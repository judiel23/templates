One Page

Structure

Inside package are presents 12 demo page.
2 of this ones are cooming soon page.
Each demo has its folder. Each folder is numbered.
Each demo has its files identified by a suffix relative to the number of demo.

PHP Version - Eg. callout-10.php it's referred to the demo 10.
Only the index page hasn't the number suffix. Each index page is named index.php and it's included in its folder.
HTML Version - All sections are splitted in one unic file.

PHP Version - Each demo has its sections. There's a lot of sections included in all demos
Inside index file of each demo are described (by require php function) a list of all sections used.

Fonts Family

In each index, the body tag has identified by an id.
Each id is referred to a specif font-family. The fonts are assigned in the demo-font.css file.

Function to Minimize

Note - these functions are active only in php version.

CSS files are declared before body tag, the JS files after the body.
All css and js files are commented because in the theme are used two php functions which minimize all scripts.
this process improve the speed of pages. There're no present any file minimized in the theme, because the minimization is made from server side.
To add more scripts inside the fuction, open the js.php file located in js folder.

Note - If you want to use all scripts separately to editing, remove the comments before the file css or js in index page.
After remove or comment the functions php around line 29 (for styles css) and around line 80 (for js scripts).

Sections

Working with HTML

The Theme includes 19 sections.
Sections can be recognized by their #ID inside page html.

Sections available

  • Header <header> - It contains the logo and menu navigations.
  • Wall #wall - Usually is the first site section. It contains a slider or a full images as background.
  • Intro #intro - Usually is an about section. It contains text, icons or images.
  • Blog #blog - Include a list of articles. It can show a grid of posts with thumb and excerpt or like feeds.
  • Callout #callout - Write an highlited message with title, text and/or icons.
  • Clients #clients - Add a carousel of logos.
  • Counters #counters - Add some services counter.
  • Features #features - Add a box for your info or list of details.
  • Map #map - Add a google map with one or more markers. (Set the map by data parameters)
  • Portfolio #portfolio - Add a grid list of projects. It can shows a grid of thumbs with captions. (No inner pages)
  • Parallax #parallax - Create a box with text or icons inside parallax background.
  • Gallery #gallery - Add a grid list of images. It can shows a grid of thumbs with captions. (Lightbox ready)
  • Quotes #quotes - Set multiple message by some people. It can show a list of quotes with thumbs or with only text.
  • Services #services - Add others box informations. It usually shows columns with text and icons.
  • Team #team - Add box for your team.
  • Price #prices - Add one or more columns prices.
  • CountDown #countdown - Add a timer to countdown (available only in demo11 and demo 12. The parameters to set it are located in js/scripts.js file around line 182 (demo 11) and 192 (demo 12).
  • Widgets #widgtes - Add an area widgets with two, three or four widgets list.
  • Footer <footer> - It contains the site copyright with a bottom to top.

Classes CSS

GRID SYSTEM

The theme included a responsive grid based on flow columns

The classes are located in framework.css

You can use these classes to insert lined up content in your page

  • .col-1-6 - Attach a width of 1/6 of the content
  • .col-1-5 - Attach a width of 1/5 of the content
  • .col-1-4 - Attach a width of 1/4 of the content
  • .col-1-3 - Attach a width of 1/3 of the content
  • .col-1-2 - Attach a width of 1/2 of the content

In adding to fit better your page are present also other grid classes as .col-2-3, .col-2-4, .col-3-4, .col-2-5, .col-3-5, .col-4-5, .col-2-6, .col-3-6, .col-4-6, .col-5-6.

SPACINGS

You can use useful classes related to padding and margin to edit your content.

These classes are located in custom.css

  • .marg-25 - attach 25px of margin to tag
  • .marg-50 - attach 50px of margin to tag
  • .marg-75 - attach 75px of margin to tag
  • .marg-100 - attach 100px of margin to tag
  • .padd-25 - attach 25px of padding to tag
  • .padd-50 - attach 50px of padding to tag
  • .padd-75 - attach 75px of padding to tag
  • .padd-100 - attach 100px of padding to tag

Other possible combinations - You can mix direction or axis y/x

  • .padd-top-25 - attach 25px of padding-top to tag
  • .padd-bott-50 - attach 50px of padding-bottom to tag
  • .margin-left-75 - attach 75px of margin-left to tag
  • .margin-right-100 - attach 100px of margin-right to tag
  • .padd-x-25 - attach 25px of padding left and right to tag
  • .padd-y-50 - attach 50px of padding top and bottom to tag
  • .marg-x-75 - attach 75px of margin left and right to tag
  • .marg-y-100 - attach 100px of margin top and bottom to tag
  • .no-marg - remove all margin at the element
  • .no-padd - remove all padding at the element

And much more..

STRUCTURE

Use differt classes to set the size of element
Here some examples..

  • .full - set width and heiht to 100%
  • .full-h - set height to 100%
  • .full-w - set width to 100%
  • .h-50 - set the height of element to 50px (at intervals of 50px up to 600px)
  • .w-50 - set the wight of element to 50px (at intervals of 50px up to 600px)

POSITIONING

Set element in position relative, absolute or static..

  • .posit-a - set an element in absolute position
  • .posit-f - set an element in fixed position
  • .posit-s - set an element in static position
  • .top-0 - put an element on top position
  • .bott-0 - put an element on bottom position
  • .right-0 - put an element on right position
  • .left-0 - put an element on left position
  • .left-50 - put an element to 50px from left position (at intervals of 50px up to 300px)
  • .bott-50 - put an element to 50px from bott position (at intervals of 50px up to 300px)
  • .right-50 - put an element to 50px from right position (at intervals of 50px up to 300px)
  • .top-50 - put an element to 50px from top position (at intervals of 50px up to 300px)
  • .z-0 - put an element in z-index 0 (it can be set from 0 to z-9)
  • .z-99 .z-999 .z-9999 - put an element in z-index 9 or 99 or 9999

And much more..

DISPLAY

Set element in display block, inline-block or flotted..

  • .disp-b - set an element in display block
  • .disp-ib - set an element in display inline-block
  • .disp-n - set an element in display none
  • .disp-i - put an element in display inline
  • .disp-t - put an element in display table
  • .disp-tc - put an element in display table-cell
  • .clear - add a clear both (usefull after flotted element)
  • .float-l - put an element in flotted mode from left
  • .float-r - put an element in flotted mode from right
  • .float-n - remove flotted mode
  • .resp-n - hidden element under 1400px (discover css/responsive.css file for more details
  • .opa-0 - add an opacity of 0% to the div (it can be set from opa-0 to opa-9)
  • .overf-h - assign an overflow hidden property at the element

BACKGROUNDS

TRANSITIONS

You can use useful classes to attach some css3 animations to your box content

The classes are located in framework.css

  • .transit - add a transition about 0.3s erase-in-out
  • .transit-fade - add a fade transition
  • .transit-left - add a transition from left
  • .transit-right - add a transition from right
  • .transit-top - add a transition from top
  • .transit-bottom - add a transition from bottom
  • .transit-bouncein - add a bounce in transition
  • .transit-flip - add a 3d transition
  • .transit-rotate - add a rotate transition
  • .transit-pulse - add a pulse transition
  • .transit-bounce - add a bounce transition

And much more..

In adding to customize your transitions, you can set some parameters inserting these data attribute in your html tag

  • data-delay="millisec" - define the duration of the transition
  • data-appear="true/false" - set the start of transition when scroll page

COUNTDOWN

The parameters to customize your cooming soon page, are described around line 182

GOOGLE MAP

To customize your map reach the map section (you can find it through its 'id') or opening map.php file (PHP Version).
You can set the address by data-lng (longitude) and data-lat (latitude) attributes
You can set the color (data-color) and the saturation (data-saturation).
In adding you can attach multiple marker positions setting the same data attributes made for the map.
Finally you can add your personal marker image putting the absolute url inside data-icon attribute.
