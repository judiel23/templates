Navigation

 

Standard Menu bar structure:

<header>
    <nav class="navbar navbar-default navbar-fixed-top" role="navigation">
        <nav class="navigation">
            ...
        </nav> <!-- end navigation -->
    </nav> <!-- End Navbar -->
</header>

 

Onepage Menu bar structure:

<header id="home">
    <nav class="navbar navbar-default navbar-fixed-top" role="navigation">
        <nav class="navigation overlay">
            ...
        </nav> <!-- end navigation -->
    </nav> <!-- End Navbar -->
</header>

 

Each menu item in Onepage navigation should have class="page-scroll" and anchor with section id:

<li><a href="#services" class="page-scroll">Services</a></li>

Portfolio

Portfolio filter has the following structure:

<div class="row mt40">
    <div class="col-md-12">
        <div class="portfolio-filter">
            <a href="#" class="filter active" data-filter="*">All</a>
            <a href="#" class="filter" data-filter=".web-design">Web Design</a>
            <a href="#" class="filter" data-filter=".print">Print</a>
            <a href="#" class="filter" data-filter=".graphics">Graphics</a>
            <a href="#" class="filter" data-filter=".mockups">Mockups</a>
        </div> <!-- end filter -->
    </div> <!-- end col -->
</div> <!-- end row -->

 

Data-filter is the name of the categories.

 

Categories of the works should be defined here <div class="col-md-4 work-item web-design mockups">

Twitter Feed

To connect you twitter feed follow these steps:

1. Go to your twitter profile Options > Widgets. Click the “Create new” button to build a new timeline widget.

2. Get the id code in address link

3. Open twitterFetcher_min.js located in /js folder with any text editor and paste the id number in line 25:

"id": '594366594521804800'

Contact Form

Open contact.php with any text editor and paste your email address in line 6. Save it.

define("__TO__", "testemail@gmail.com");
